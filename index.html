Preguntas frecuentes

¿Cómo reemplazo enlaces por mis links de afiliado?
Busca dentro del código '{{AFF_LINK}}' o '{{PAYPAL_LINK}}' y reemplazalos por tus enlaces reales. Si no sabés, te dejo ejemplos en el README del repo.
¿Puedo añadir más productos automáticamente?
Sí — podés conectar un archivo JSON en el mismo repo y cambiar la carga en el script para hacer fetch('/productos.json').
